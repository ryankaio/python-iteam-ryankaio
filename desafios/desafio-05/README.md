# Desafio 05 — Gerenciador de Compras

**Aula:** 05 | **Tema:** Listas e Estruturas de Dados
**Prazo:** até o final da Aula 05

---

## Enunciado

Crie um script `solucao.py` que:
1. Comece com uma lista vazia.
2. Use um laço `while` para pedir ao usuário nomes de produtos.
3. Adicione cada produto à lista usando `append`.
4. Pare quando o usuário digitar "fim".
5. Ao final, exiba a lista organizada e o total de itens com `len()`.

---

## Pergunta Obrigatória (responda em `explicacao.md`)

> Por que usamos uma **lista** e não uma **tupla** para essa lista de compras? O que mudaria no comportamento do programa se tentássemos usar tupla?
