## Resposta à Pergunta Obrigatória

> Por que usamos uma **lista** e não uma **tupla** para essa lista de compras? O que mudaria no comportamento do programa se tentássemos usar tupla?

_(Usamos uma lista porque a lista de compras precisa ser modificada durante a execução do programa. A cada produto digitado pelo usuário, adicionamos um novo item com append(). As listas em Python são mutáveis, ou seja, seus elementos podem ser adicionados, removidos ou alterados depois que a lista é criada, Já as tuplas são imutáveis. Depois de criadas, não é possível adicionar, remover ou alterar elementos diretamente.)_
