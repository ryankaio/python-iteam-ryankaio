## O que meu programa faz

_(No arquivo funcoes_mat.py, são definidas três funções matemáticas: uma para calcular a área de um círculo, outra para calcular o volume de uma esfera e uma terceira para calcular a hipotenusa de um triângulo retângulo. Cada função recebe os valores necessários, realiza o cálculo usando fórmulas matemáticas e retorna o resultado.
No arquivo solucao.py, que é o programa principal, essas funções são importadas e disponibilizadas para uso. Ao ser executado, o programa exibe um menu com três opções de cálculo. O usuário escolhe a operação desejada e informa os valores necessários, como o raio ou os catetos do triângulo. Em seguida, o programa chama a função correspondente, realiza o cálculo e exibe o resultado formatado na tela. Caso o usuário digite uma opção inexistente, o sistema informa que a escolha é inválida.)_

## Resposta à Pergunta Obrigatória

> Por que separar as funções em um arquivo diferente do `solucao.py`? O que muda no projeto quando você tem 50 funções em vez de 3?

_(O programa foi dividido em dois arquivos para aplicar o conceito de modularização, que consiste em separar funções e responsabilidades em módulos diferentes, tornando o código mais organizado e reutilizável.)_
