# Desafio 04 — Tabuada Personalizada

**Aula:** 04 | **Tema:** Estruturas de Repetição
**Prazo:** até o final da Aula 04

---

## Enunciado

Crie um script `solucao.py` que:
1. Solicite ao usuário um número de 1 a 10.
2. Utilize um laço `for` para exibir a tabuada desse número.
3. Garanta que o programa pare se o usuário digitar 0.
4. **Bônus:** Use um `while` para permitir várias tabuadas sem reiniciar o script.

---

## Pergunta Obrigatória (responda em `explicacao.md`)

> Para esse exercício, por que `for` com `range()` é preferível ao `while`? Em que cenário o `while` seria a escolha certa?
